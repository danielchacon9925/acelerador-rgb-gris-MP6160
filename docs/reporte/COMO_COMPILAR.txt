COMPILACIÓN DEL REPORTE
=======================

Requisitos (Debian/Ubuntu):

    sudo apt install texlive-latex-recommended texlive-lang-spanish \
                     texlive-latex-extra texlive-fonts-recommended lmodern

Compilar (dos pasadas, para que el índice quede bien):

    pdflatex reporte_MP6160.tex
    pdflatex reporte_MP6160.tex

O bien:

    latexmk -pdf reporte_MP6160.tex

Limpiar auxiliares:

    latexmk -c


QUÉ FALTA COMPLETAR
===================

Cada bloque amarillo "PENDIENTE" marca una sección por completar.
Localizarlos con:

    grep -n "pendiente{" reporte_MP6160.tex
